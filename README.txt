[AT Lab#19]JS

Help:
	node index.js --help
	node index.js -h
	
Usage:
	node index.js --id [string] --type [string]

Options:
	--version        Show version number                                 [boolean]
	-i, --id         Number of ID
	-n, --name       Name of character
	--st, --status   Status: alive or dead
	--sp, --species  Species of character(e.g. Human)
	-t, --type       Type
	-g, --gender     Gender: male or female
	-o, --origin     Origin(e.g. Earth)
	-l, --location   Location(e.g. Earth)
	-h, --help       Show help                                           [boolean]

Examples:
	index.js -n Alex
	index.js --st Alive
